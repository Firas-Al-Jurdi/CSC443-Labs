Google Fonts is a library that includes free and open source font families, an interactive web
directory for browsing the library, and APIs for using the fonts via CSS and Android.
Popular fonts in the Google Fonts library include Roboto, Open Sans, Lato, Oswald,
Montserrat, Source Sans Pro, and Raleway.
